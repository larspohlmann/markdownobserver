# Plan: Cache Layer Redesign

> Generated by Claude Code after profiling memory usage in `SyncCoordinator`

## Problem

`NSCache` backend grows unbounded and isn't safe under strict concurrency.

## Proposed Architecture

```swift
actor CacheManager<Key: Hashable & Sendable, Value: Sendable> {
    private let memoryLimit: Int
    private var entries: [Key: CacheEntry<Value>] = [:]

    func get(_ key: Key) -> Value? {
        guard var entry = entries[key] else { return nil }
        entry.lastAccessed = Date()
        entries[key] = entry
        return entry.value
    }

    func set(_ key: Key, value: Value, size: Int) {
        evictIfNeeded(requiredSpace: size)
        entries[key] = CacheEntry(value: value, size: size, createdAt: .now)
    }
}
```
