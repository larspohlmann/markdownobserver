# Security Policy

## Supported Versions

Security fixes are applied to the latest version on the default branch.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately.

- Preferred: open a private GitHub Security Advisory draft for this repository:
	https://github.com/larspohlmann/markdownobserver/security/advisories/new
- Maintainer contact: Lars Pohlmann on GitHub:
	https://github.com/larspohlmann
- Please do not open public issues for security vulnerabilities.

Please include:

- A clear description of the issue.
- Reproduction steps or a proof of concept.
- Expected impact.
- Any suggested mitigations.

We will acknowledge receipt as quickly as possible and coordinate next steps, including disclosure timing.
